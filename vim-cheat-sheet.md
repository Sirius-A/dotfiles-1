# VIM Cheat Sheet

`<C-w>f` - file under cursor open in new horzontial split  
`<C-w>gf` - file under cursor open in new tab

`gf` - open file under cursor  
`gx` - open weblink under cursor   
                                                                                 
`zz` - scroll center   
`zt` - scroll top                                                                 
`zb` - scroll bottom                                                              
                                                                                 
`H` - cursor top                                                                  
`M` - cursor middle                                                               
`L` - cursor bottom                                                               
                                                                                 
`gqq` - wrap a 80                                                                 
`gq{some sign}` - wrap till {some sign}
